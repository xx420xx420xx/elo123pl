Instalacja 999cape

Windows

Pobierz

Manualna instalacja

Przejdź do nastepującego folderu: C:\Windows\System32\drivers\etc
a następnie zedytuj plik hosts dopisując na samym końcu:
2.59.134.131 s.optifine.net

macOS

Otwórz terminal i wpisz poniższą komendę:

curl https://999cape.art/macos_installation.sh | bash -s install

Linux

Otwórz terminal i wpisz poniższą komendę:

wget -O - https://999cape.art/linux_installation.sh | bash -s install
W razie problemów z instalacją utwórz ticket na naszym serwerze Discord!
